# gabrielgerbi.com.br

Meu portfólio — Full Stack Developer

## Stack

- **Next.js 14** — App Router
- **TypeScript**
- **Tailwind CSS**

## Rodar localmente

```bash
npm install
npm run dev
```

## Deploy

Push para o GitHub e conecte ao Vercel.
